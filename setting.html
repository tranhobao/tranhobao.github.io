Cách cài đặt

Quá trình cài đặt sẽ hơi khó khăn, bạn cố gắng nhé

1. Cài đặt python3

Copy 
        sudo apt-get install python3

2. Cài đặt youtube-dl

Copy 
        sudo pip install youtube-dl

3. Chỉnh sửa đường dẫn

        Vào đường dẫn cài đặt python3: usr/lib/python3/dist-package/youtube-dl/extractor/youtube.py

Chỉnh sửa từ - Before

Copy 
        'uploader_id': self._search_regex(r'/(?:channel|user)/([^/?&#]+)', owner_profile_url, 'uploader id')
         if owner_profile_url else None

Chỉnh sửa thành - After

Copy 
        'uploader_id': self._search_regex(r'/(?:channel|user)/([^/?&#]+)', owner_profile_url, 'uploader id' , 
        fatal=False ) if owner_profile_url else None

4. Cài đặt tkinter

Copy 
        sudo apt-get install python3-tk
