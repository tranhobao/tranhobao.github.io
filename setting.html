Cách cài đặt Windows

Quá trình cài đặt sẽ hơi khó khăn, bạn cố gắng nhé

1. Tải xuống gói cài đặt Python

        Bước 1: Cài đặt Python tại trang chủ:
        https://www.python.org/downloads/
        
        Chọn mục Windows để lựa chọn file cài đặt phù hợp với máy tính của mình
      
        Bước 2: Download Python - hiện tại phiên bản mới nhất là 3.9.5:
        
      
        Bước 3: Chọn gói cài đặt sao cho phù hợp với máy tính của bạn. Nếu máy tính 64-bit thì chọn gói cài cho 64-bit, 
        nếu máy tính 32-bit thì chọn gói cài cho 32-bit.

2. Cài đặt Python

        Bước 1: Điều hướng đến thư mục chứa bộ cài python mà bạn đã tải xuống trước đó. Nhấn đúp chuột trái để thực hiện quá trình cài đặt. 
        Hệ thống có thông báo xác nhận cài đặt hay không, bạn hãy chọn Yes để chấp thuận cài đặt python cho máy (nếu có).
      
        Bước 2: Cửa sổ hiển thị quá trình cài python hiện lên. Tại đây bạn hãy nhớ đánh dấu Tick vào tùy chọn Add Python 3.9 to PATH để bộ cài tự động thêm biến môi trường cho máy tính. 
        Phiên bản mình đã tải xuống là 3.9 nên tùy chọn về biến môi trường là 3.9.
        
      
        Bước 3: Tại đây, click Next và Install.
        
        
        
      
        Bước 4: Cửa sổ thông báo cài đặt thành công, bạn click Close để chuyển sang viết chương trình thử nghiệm đầu tiên nhé.

3. Chỉnh sửa đường dẫn

        Vào đường dẫn cài đặt python3: C:\Users\'Tên Users'\AppData\Local\Programs\Python\Python311\Lib\site-packages\youtube_dl\extractor\youtube.py

Chỉnh sửa từ - Before

Copy 
        self._search_regex(r'/(?:channel|user)/([^/?&#]+)', owner_profile_url, 'uploader id') if owner_profile_url else None

Chỉnh sửa thành - After

Copy 
        'After: 'uploader_id': self._search_regex(r'/(?:channel|user)/([^/?&#]+)', owner_profile_url, 'uploader id' , fatal=False ) if owner_profile_url else None
